Memoria

Descripción de la práctica

En esta sesión práctica, hemos explorado WebRTC para la comunicación en tiempo real a través del navegador.

Ejercicio 1

En este ejercicio hemos creado una página web donde el flujo de la cámara se muestra directamente en un elemento de vídeo, utilizando la API getUserMedia.

Ejercicio 2

En este ejercicio hemos ampliado esta funcionalidad permitiendo capturar imágenes del vídeo en vivo y guardarlas como archivos, simulando el funcionamiento de una cámara fotográfica.

Ejercicio 3

Aqui implementamos una videollamada local mediante RTCPeerConnection, estableciendo la comunicación entre dos extremos dentro de la misma página. Para ello, configuramos el intercambio de descripciones multimedia y utilizamos el protocolo ICE para facilitar la conexión.

Ejercicio 4

Finalmente, integramos WebRTC con WebGL, modificando una escena en Three.js (prac3-2.html) para que la textura de un plano utilizara el vídeo capturado en tiempo real, en lugar de un archivo pregrabado.
